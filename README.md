# Portfolio Agency - React SPA

A high-performance, visually stunning Portfolio Agency landing page built with **React 19** and **Bun**. This application features advanced animations and a modular component architecture.

## 🚀 Features

- **Premium GSAP Animations**: Integrated `ScrollTrigger`, `ScrollSmoother`, and `SplitText` for fluid, interactive motion.
- **Fast Runtime**: Powered by **Bun** for lightning-fast builds and hot reloading.
- **Responsive Design**: Mobile-first approach with custom Tailwind CSS styling.
- **Modular Components**: Cleanly separated UI sections (Hero, About, Projects, Blog, etc.).
- **Custom Counter**: A lightweight, dependency-free intersection-based counter implementation.

## 🛠️ Built With

### Core
- **React 19**: Modern UI library.
- **Bun**: JavaScript runtime and package manager.
- **TypeScript**: Static typing for robust development.

### Styling & UI
- **Tailwind CSS**: Utility-first styling framework.
- **Lucide React**: Clean and consistent iconography.
- **Radix UI**: Accessible headless UI primitives.

### Animation
- **GSAP (GreenSock)**: The industry standard for robust web animations.
  - `@gsap/react`: React-specific GSAP hooks.
  - `ScrollTrigger`: Scroll-based animation control.
  - `ScrollSmoother`: Smooth scroll experience.
  - `SplitText`: Professional text manipulation.

## 🏃 How to Run

### 1. Install Dependencies
```bash
bun install
```

### 2. Start Development Server
```bash
bun run dev
```

### 3. Build for Production
```bash
bun run build
```

The application will be available at `http://localhost:3000` (or the port specified by Bun).
